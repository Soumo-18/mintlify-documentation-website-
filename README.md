# Mintlify Documentation Website Clone

A pixel-perfect recreation of the [Mintlify](https://mintlify.com/) landing page, built using pure HTML and CSS. This project focuses on structure, layout accuracy, and clean design implementation without the use of frameworks or JavaScript.

**Original Design Reference:** [Mintlify Brand Assets](https://brandfetch.com/mintlify.com)

## üöÄ How to Open Locally

Follow these steps to run the project on your local machine:

1.  **Clone the repository:**
    Open your terminal or command prompt and run:
    ```bash
    git clone [https://github.com/Soumo-18/mintlify-documentation-website-.git](https://github.com/Soumo-18/mintlify-documentation-website-.git)
    ```

2.  **Navigate to the project folder:**
    ```bash
    cd mintlify-documentation-website-
    ```

3.  **Open the website:**
    * **Option A (Double-click):** Locate the `index.html` file in the folder and double-click it to open it in your default browser.
    * **Option B (VS Code Live Server):** If you use VS Code, right-click `index.html` and select "Open with Live Server".
